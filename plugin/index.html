Duration Priority Formula

How task duration affects priority. Inverse = shorter tasks get higher priority.

Duration Priority = linear(hours) * 1.0
Oldness Priority Formula

How task age affects priority. Older tasks can get higher priority to prevent them from being forgotten.

Oldness Priority = linear(days) * 1.0
Total Priority Formula
Total Priority = Base(N+1-rank) + TagBoosts + DurationPriority + OldnessPriority

How Merging Works
  • Merging combines all incomplete splits of a task
  • Time estimates are summed together
  • The first incomplete split becomes the merged task
  • Other splits are marked as completed
  • The original task title is restored (without Roman numerals)
